Key Features

🔍

Automated Audits

Run Google Lighthouse audits on any URL with a single API call. Get scores for Performance, Accessibility, Best Practices, and SEO.

📊

Real-Time Dashboard

Beautiful dark-mode dashboard built with Next.js 14 and Recharts. View performance trends and critical issues at a glance.

⚡

Distributed Job Queue

BullMQ-powered Redis queue with automatic retries, exponential backoff, and concurrent processing. Scale horizontally.

🐳

One-Command Deploy

Full Docker Compose stack — PostgreSQL, Redis, Web App, and Worker containers. From zero to running in 2 mins.

📈

Historical Trends

Track performance over time with interactive charts. Identify regressions, measure improvements, and set baselines.

🔔

Status Monitoring

Real-time worker health checks, job status tracking (Pending → Processing → Completed), and pass/fail indicators.

Quick Start

Deploy quickly with Docker Compose

terminal 
# 1. Clone the repository
git clone https://github.com/lekhanpro/perfpatrol.git
cd perfpatrol

# 2. Configure environment
cp .env.example .env

# 3. Start the entire stack
docker-compose up -d

# 4. Open http://localhost:3000
