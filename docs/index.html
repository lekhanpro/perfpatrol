Why Perf-Patrol?

Everything you need to monitor, analyze, and improve web performance.

Automated Audits

Run Google Lighthouse audits on any URL with a single API call. Get scores for Performance, Accessibility, Best Practices, and SEO.

Real-Time Dashboard

Beautiful dark-mode dashboard built with Next.js 14 and Recharts. View performance trends and critical issues at a glance.

Distributed Job Queue

BullMQ-powered Redis queue with automatic retries, exponential backoff, and concurrent processing. Scale horizontally with ease.

One-Command Deploy

Full Docker Compose stack — PostgreSQL, Redis, Web App, and Worker containers. From zero to running in under 2 minutes.

Historical Trends

Track performance over time with interactive charts. Identify regressions, measure improvements, and set baselines for your sites.

Worker Health Monitoring

Real-time worker health checks, job status tracking (Pending → Processing → Completed), and pass/fail indicators.

Simple REST API

One endpoint to trigger audits. Clean JSON responses.

POST /api/scan

Submit a URL for Lighthouse audit. Returns audit ID for tracking.

Request Body 
{
  "url": "https://example.com",
  "projectId": "proj_01",
  "projectName": "My Website",
  "frequency": "daily"
}
200 Response

Successful response with audit and project identifiers.

JSON Response 
{
  "success": true,
  "auditId": "clx1abc...",
  "projectId": "proj_01"
}

Quick Start

Deploy the full stack in under 2 minutes with Docker Compose.

1

Clone the repository

$ git clone https://github.com/lekhanpro/perfpatrol.git $ cd perfpatrol
2

Configure environment

$ cp .env.example .env
3

Start the entire stack

$ docker-compose up -d
4

Open the dashboard

# Visit http://localhost:3000

Flexible Deployment

Choose the deployment strategy that fits your needs.

Full-stack deployment with PostgreSQL, Redis, Web App, and Worker containers orchestrated together.
